Steller-Worlds - Version 0.1

Welcome to the Steller-Worlds website project! This is a simple, responsive HTML and CSS-based website designed to serve as a starting point for a broader web application. Currently, the website features a basic layout with a sticky navigation bar and some initial styling.

Features
- Responsive Design: The website adapts to various screen sizes from mobile devices to 8K monitors.
- Sticky Navigation Bar: The top navigation bar stays fixed at the top of the screen as you scroll.
- Hover Effects: Links in the navigation menu change style when hovered over.

Technologies Used
- HTML: The structure of the website is created using semantic HTML.
- CSS: Styling and layout are managed using external CSS with various media queries for responsiveness.
- Responsive Media Queries: The website scales for different screen resolutions (1080p, 2K, 4K, and 8K).
